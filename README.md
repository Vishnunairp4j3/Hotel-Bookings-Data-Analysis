# Hotel-Bookings-Data-Analysis

# Project Summary

This project analyzes hotel booking data to uncover patterns in customer behavior, booking trends, cancellations, revenue generation, and seasonal demand. The analysis follows a complete data analytics workflow, including data cleaning, preprocessing, exploratory data analysis (EDA), visualization, and business insight generation.

### Objectives

* Understand hotel booking patterns and customer behavior.
* Identify factors influencing booking cancellations.
* Analyze revenue performance across hotel types and market segments.
* Examine seasonal trends and demand fluctuations.
* Generate actionable business recommendations to improve occupancy and profitability.

### Project Workflow

#### 1. Data Loading and Initial Exploration

* Imported and inspected the dataset.
* Reviewed dataset structure, data types, and statistical summaries.
* Explored key variables and identified data quality issues.

#### 2. Data Preprocessing

* Handled missing values.
* Removed duplicate records.
* Corrected data types.
* Created derived features for enhanced analysis.
* Filtered anomalies and prepared a clean dataset.

#### 3. Exploratory Data Analysis (EDA)

* Analyzed booking distributions and customer segments.
* Investigated cancellation drivers.
* Examined revenue performance using Average Daily Rate (ADR).
* Explored seasonal booking trends.
* Studied guest demographics and repeat customer behavior.
* Performed correlation analysis between key variables.

#### 4. Data Visualization

Created multiple visualizations including:

* Hotel type distribution and cancellation rates
* Monthly booking trends
* Market segment analysis
* Lead time distribution
* ADR distribution
* Deposit type analysis
* Booking heatmaps
* Correlation heatmaps
* Top guest countries
* Seasonal revenue analysis
* Customer behavior visualizations

### Key Insights

* Cancellation rates vary significantly across booking channels and deposit types.
* Lead time has a strong relationship with booking cancellations.
* Seasonal demand impacts occupancy and revenue generation.
* Different hotel types attract distinct customer segments.
* Repeat guests exhibit different booking behaviors compared to new customers.

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

### Business Impact

The findings from this analysis can help hotel management optimize pricing strategies, reduce cancellation rates, improve customer retention, forecast demand more accurately, and maximize overall revenue.
